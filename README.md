# 프로젝트 소개   
- ### 코로나19 펜데믹으로 인한 **운동부족을 산책**을 통해 해결하고자 함.
- ### 산책은 **비용이 들지 않으며,** 코로나19 회복을 위해 **유산소 운동은 필연적**임.
- ### 현재 시장에서 **산책관련 어플이 부족함.**
- ### **산책을 통한 건강 증진 및 주변 이용자와의 소통으로 함께 걷는 산책을 목표로 함.**
